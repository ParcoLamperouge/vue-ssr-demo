# ssr-demo1

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
### Compile as server render

```
npm run build:win
node server.js
```

### Run on server
copy folder to your server
```
npm install -g pm2
npm install
pm2 server.js
```
